# DEVIK DIGITAL SOLUTIONS - Company Profile

This project is a company profile web page for **DEVIK DIGITAL SOLUTIONS**. It includes the cover, about section, services, capabilities, industries served, testimonials, and export options.

## Getting Started

```bash
npm install
npm run dev
```

## High-Quality PDF Export (Headless Chrome)

This preserves the exact design and layout.

1. Start the dev server in one terminal:

```bash
npm run dev
```

2. In another terminal, run:

```bash
npm run export:pdf
```

This will save `DEVIK_DIGITAL_SOLUTIONS_PROFILE.pdf` in the project root.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Notes

- Use the in-app buttons to export as **PPTX** or **PDF**.
